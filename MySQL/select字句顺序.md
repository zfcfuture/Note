### SELECT字句顺序



|   字句   |       说明       |      是否必须使用      |
| :------: | :--------------: | :--------------------: |
|  SELECT  | 要返回的列或表达 |           是           |
|   FROM   | 从中检索数据的表 | 仅在从表选择数据时使用 |
|  WHERE   |     行级过滤     |           否           |
| GROUP BY |     分组说明     | 仅在按组计算聚集时使用 |
|  HAVING  |     组级过滤     |           否           |
| ORDER BY |   输出排序顺序   |           否           |
|  LIMIT   |   要检索的行数   |           否           |
